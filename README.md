# result-predictor
# Student Result Prediction Using Machine Learning

##  Project Overview

This project is a Machine Learning based web application that predicts whether a student will **Pass** or **Fail** based on input values such as:

* Study Hours
* Attendance Percentage
* Marks

The model is trained using a classification algorithm and deployed using Flask.

---

##  Features

* Predicts student result instantly
* User-friendly web interface
* Machine Learning classification model
* Fast and accurate predictions
* Simple and responsive design

---

##  Technologies Used

* Python
* Pandas
* Scikit-learn
* Flask
* HTML
* CSS
